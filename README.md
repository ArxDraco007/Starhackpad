# Starhackpad

## Why did I do this project?
If you use a computer, tablet, or phone (I don't know what you are using otherwise). A macropad is one of the best add-ons you can have to increase your productivity. A macropad is a small, compact, secondary keypad with fully programmable keys. It allows you to execute complex keyboard shortcuts, automate repetitive tasks, or launch applications with a single press of a button. One press of a button can run the commands to curse at your friends when they are playing like they usually would.
